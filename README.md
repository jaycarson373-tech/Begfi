# BegFi

BegFi is a polished Next.js/Tailwind landing page and dashboard concept for a CT-native Beg-To-Earn protocol on Solana.

## Stack

- Next.js app router
- Tailwind CSS
- Framer Motion
- Lucide icons
- Launch-ready feed and dashboard placeholders

## Feed Boundary

The feed and dashboard are ready for the X scanner, wallet verification, voting, and reward distribution services once those launch.

## Commands

```bash
pnpm install
pnpm dev
pnpm build
```
